# 📚 RAG PDF Chatbot

A Retrieval-Augmented Generation (RAG) app that answers questions from a PDF.

## 🚀 Features
- Ask questions from PDF
- FAISS vector database
- HuggingFace embeddings
- Groq LLM (fast inference)
- Gradio UI

## 🛠 Tech Stack
- LangChain
- FAISS
- Sentence Transformers
- Groq
- Gradio

## 🔐 Setup

1. Open notebook in Google Colab
2. Add your Groq API key in Colab Secrets:
   - Key: GROQ_API_KEY
3. Run the cell

## 📂 Input
- Public Google Drive PDF link (https://drive.google.com/file/d/1Q4EUHwH9yotWAR3S9QaktvAI-tV6qFW3/view?usp=drive_link)
